# Traffic Sign Detection and Recognition

## Overview
This project focuses on building a computer vision system capable of detecting and recognizing traffic signs in real time. The aim is to improve road safety and assist in applications such as autonomous driving and driver assistance systems.

The model is trained on a labeled dataset of traffic signs and uses deep learning techniques to identify and classify signs from images and live video input.

---

## Objectives
- Detect traffic signs in images and video streams
- Classify detected signs into their respective categories
- Build a real-time system using webcam input
- Improve model accuracy through preprocessing and structured data handling

---

## Tech Stack
- Python
- TensorFlow / Keras
- OpenCV
- YOLO (for detection)
- Kaggle (for training and experimentation)

---

## Dataset
The dataset used contains multiple classes of traffic signs with labeled images. It follows a structured format suitable for training object detection models. Data preprocessing steps were applied to ensure consistency and improve model performance.

---

## Methodology
- Organized dataset following proper YOLO format
- Applied preprocessing techniques such as resizing and normalization
- Trained the model using deep learning architectures
- Implemented real-time detection using webcam input
- Evaluated performance based on prediction accuracy and detection capability

---

## Features
- Real-time traffic sign detection using webcam
- Multi-class classification of traffic signs
- Bounding box detection for precise localization
- Scalable pipeline for further improvements

---

## Challenges Faced
- Handling variations in image resolution and lighting
- Reducing misclassification between similar signs
- Improving detection speed without compromising accuracy

---

## Results
The model performs well on standard test data and is capable of recognizing most traffic signs with good accuracy. Performance improves with better dataset quality and proper training configuration.
